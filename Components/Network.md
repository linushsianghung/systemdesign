```markdown
- MAC (Media Access Control) Address: A unique, hardware-based identifier for a device's network interface card (NIC). It's a global identifier so it's very difficult to find the target in a huge network space.
- IP (Internet Protocol) Address: With `Network` and `Host` sections, total **4 bytes**, the can target can be easier to find by specifying the Network section. And then **Hosts in the same Network can use MAC address as usual.**
- ARP (Address Resolution Protocol): Used to map a logical IP address to a physical MAC address within a local area network (LAN). **It has to be used in the same network.**
```
